# App de Clima

## Descripción

Esta aplicación web muestra información climática básica de distintas ciudades de Chile.  
Cuenta con una vista principal donde se listan las ciudades disponibles y una vista de detalle donde se presenta el pronóstico semanal junto con estadísticas simples.

La aplicación utiliza datos simulados (mock) y no depende de APIs externas ni de un servidor para su ejecución.

---

## Funcionalidades

- Listado de ciudades disponibles
- Navegación a vista de detalle por ciudad
- Visualización de pronóstico semanal
- Cálculo de estadísticas:
  - Temperatura máxima
  - Temperatura mínima
  - Promedio semanal
- Interfaz básica con estilos CSS

---

## Tecnologías utilizadas

- HTML5
- CSS3
- JavaScript (ES5/ES6)
- Manipulación del DOM
